# Emotion Aware Study Focus Analyzer

## Overview
Emotion Aware Study Focus Analyzer is a desktop-based machine learning application that monitors and analyzes a student's emotional state and focus level during study sessions using real-time facial expression recognition.

The system uses computer vision and deep learning techniques to detect emotions, track distractions, and generate productivity reports.

## Features
- Real-time face detection
- Emotion recognition
- Focus monitoring
- Distraction tracking
- Productivity scoring
- Session-based reports
- Local processing for privacy

## Technologies Used
- Python
- OpenCV
- Machine Learning
- CNN-based Emotion Detection
- ONNX Runtime
- Computer Vision

## Project Structure
- main.py – Main application
- brain_router.py – Processing logic
- utils/ – Utility modules
- emotion-ferplus-8.onnx – Emotion detection model
- requirements.txt – Dependencies
